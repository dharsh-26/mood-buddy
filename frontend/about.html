About Mood Buddy

Mood Buddy is a Full Stack Web Application designed to help students manage their daily mood, organize study plans, and track learning progress.

✨ Features

  • 😊 User Registration & Login
  • 📅 Create, Update and Delete Study Plans
  • 📊 Track Study Progress
  • 😀 Daily Mood Tracker
  • 👤 Update User Profile
  • 🔒 Secure Authentication using JWT

💻 Technologies Used

HTML5
Bootstrap 5
JavaScript
Node.js + Express.js + MongoDB

🎯 Project Objective

The goal of Mood Buddy is to encourage productive study habits by allowing users to record their mood, plan study sessions, monitor progress, and manage their profile through a secure and user-friendly interface.

Developed as a Full Stack Development Project using the MERN backend technologies.
